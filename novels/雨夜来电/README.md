## 阅读建议

- **第一次阅读**：按第1–28章顺序读正文即可，故事完整闭环。
- **意犹未尽**：继续读第29–63章番外，含蜜月、育儿、十年、角色外传等。
- **一句话概括**：投行分析师林晚与资本大佬陆景深，因一通凌晨错拨电话重逢，在雨夜与误会里，把「我的女孩」「我的男人」写成一辈子。

## 简介

投行分析师林晚与并购大佬陆景深，曾在上上海的雨夜里相爱，又在误会中分手三年。一通凌晨错拨的电话，将两个人重新拽回同一条轨道上。职场对峙、旧情复燃、家族阻挠、生死考验——这一次，他们能否把没说完的话、没做完的事，全部交代清楚？
