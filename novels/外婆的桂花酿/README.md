## 简介

被父母留在江南小镇的女孩沈宛，由外婆一手带大。外婆每年秋天用满院桂花酿一坛糖渍，那坛甜，是她童年全部的光。后来她被接进城、长大、远走，桂花酿成了每年从故乡寄来的一个邮包。直到外婆老去、记忆一点点褪色，最后一个秋天，那坛桂花还没来得及酿成——沈宛终于回到老屋，亲手续上了那坛迟到了很多年的甜。

一坛桂花酿，酿的是一个老人沉默而绵长的爱，也是一个孩子用半生才读懂的、关于"来不及"的功课。

## 人物

- **沈宛**：女主人公。三岁被送到外婆身边，小名"阿宛"。
- **外婆（周秀英）**：江南小镇上的老人，沉默、要强、手巧，会酿桂花。
- **外公（沈长庚）**：在沈宛出生前几年去世，只活在外婆的只言片语与一张旧照片里。
- **沈宛母亲（周丽）**：年轻时进城打拼，与女儿、母亲都有亏欠。
- **王伯、豆腐西施、阿福**：小镇邻里，构成沈宛童年的人情底色。
