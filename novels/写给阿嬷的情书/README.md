## 简介

以书信体，写给已逝的祖母"阿嬷"。借番薯粥、门口的灯、粗糙的手、乡音里的小名等日常细节，铺陈祖孙之间未曾言说的爱，以及"懂事太晚"的遗憾与思念。情感克制而绵长，结尾以方言唤名收束，余韵留白。

## 创作笔记

- **书信体**：以"阿嬷"为倾诉对象，第二人称贯穿，拉近距离感。
- **以小见大**：不写大事件，只写一碗粥、一只鞋、一盏灯、一片鹅肉，用具体可感的细节承载情感。
- **方言点缀**："食饱未""转来了""莫等门"等乡音，强化乡愁与真实感（潮汕/客家语感，非特定方言考据）。
- **留白收束**：结尾不点破生死，以路灯再亮一下作意象呼应，把哀而不伤的余韵交给读者。
