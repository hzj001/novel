# 流星帝国

### —— 隋朝兴亡三十七年

> 一部用通俗、幽默笔法讲述隋朝（581—618）兴亡的历史小说。
>
> 本书为原创作品，叙述风格借鉴了"严肃历史 + 轻松讲述"的路数，
> 但书名、文字、结构、议论均为独立创作，未抄袭任何已有著作。
> 史实大体依据正史框架，细节处为增强可读性有合理的文学加工，
> 读者若想做严肃考据，请以《隋书》《资治通鉴》等原始史料为准。

## 内容简介

公元581年，一个叫杨坚的男人，从自己外孙手里"接过"了天下，建立了隋朝。

短短九年后，他灭掉南方的陈朝，结束了从西晋末年开始、长达近三百年的大分裂。
这是一件了不起的事——了不起到我们今天都低估了它。

又过了二十多年，他的儿子杨广，一个智商极高、精力极旺、野心极大的皇帝，
用最快的速度，把这个看似坚不可摧的帝国，开成了一辆失控的快车，
然后连人带车，一起冲下了悬崖。

从大一统到灰飞烟灭，隋朝只用了三十七年。

它像一颗划过夜空的流星，亮得刺眼，灭得也干脆。
可它留下的东西——科举、三省六部、大运河——却撑起了之后将近一千四百年的中国。

这是一个关于野心、关于人性、关于"步子迈太大"的故事。
